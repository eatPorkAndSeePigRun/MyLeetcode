# 136. Single Number

## Description:

Given a `non-empty` array of integers, every element appears twice except for one. Find that single one.

## Solution:

```c++
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int res = 0;
        for (auto &num : nums)
            res ^= num;
        return res;
    }
};
```
