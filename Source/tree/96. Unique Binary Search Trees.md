# 96. Unique Binary Search Trees

## Description:

Given *n*, how many structurally unique **BST's** (binary search trees) that store values 1 ... *n*?

## Solution:

```c++
class Solution {
public:
    int numTrees(int n) {
        long res = 1;
        for (int i = 0; i < n; ++i)
            res = res * 2 * (2 * i + 1) / (i + 2);
        return res;
    }
};
```
