# 242. Valid Anagram

## Description:

Given two strings `s` and `t` , write a function to determine if `t` is an anagram of `s`.

## Solution:

```c++
class Solution {
public:
    bool isAnagram(string s, string t) {
        if (s.size() != t.size())
            return false;

        vector<int> count(26, 0);
        for (auto &c : s)
            ++count[c-'a'];
        for (auto &c : t) {
            --count[c-'a'];
            if (count[c-'a'] < 0)
                return false;
        }
        return true;
    }
};
```
