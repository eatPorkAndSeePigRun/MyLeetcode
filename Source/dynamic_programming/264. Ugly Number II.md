# 264. Ugly Number II

## Description:

Write a program to find the `n`-th ugly number.

Ugly numbers are **positive numbers** whose prime factors only include `2, 3, 5`. 

## Solution:

```c++
class Solution {
public:
    int nthUglyNumber(int n) {
        vector<int> nums(n, 0);
        nums[0] = 1;
        
        int i2 = 0;
        int i3 = 0;
        int i5 = 0;
        for (int i = 1; i < n; ++i) {
            int next2 = nums[i2] * 2;
            int next3 = nums[i3] * 3;
            int next5 = nums[i5] * 5;
            int next = min({next2, next3, next5});
            
            if (next == next2) ++i2;
            if (next == next3) ++i3;
            if (next == next5) ++i5;
            nums[i] = next;
        }
        
        return nums[n-1];
    }
};
```
