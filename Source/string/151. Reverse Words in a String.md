# 151. Reverse Words in a String

## Description:

Given an input string, reverse the string word by word.

## Solution:

```c++
class Solution {
public:
    string reverseWords(string s) {
        istringstream in(s);
        string res, word;
        while (in >> word) 
            res = word + " " + res;
        res.pop_back();
        return res;
    }
};
```
