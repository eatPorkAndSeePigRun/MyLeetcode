# 13. Roman to Integer

## Description:

Given a roman numeral, convert it to an integer. Input is guaranteed to be within the range from 1 to 3999.

## Solution:

```c++
class Solution {
public:
    int romanToInt(string s) {
        if(s.empty())
            return 0;
        
        unordered_map<char, int> char2int;
        char2int['I'] = 1;
        char2int['V'] = 5;
        char2int['X'] = 10;
        char2int['L'] = 50;
        char2int['C'] = 100;
        char2int['D'] = 500;
        char2int['M'] = 1000;
        
        int res = 0, len = s.size();
        int pre = char2int[s[0]], cur = 0;
        for (int i = 0; i < len; ++i) {
            cur = char2int[s[i]];
            if (cur > pre)
                res -= 2*pre;
            res += cur;
            pre = cur;
        }
        return res;
    }
};
```
