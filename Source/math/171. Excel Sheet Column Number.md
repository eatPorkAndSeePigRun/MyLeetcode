# 171. Excel Sheet Column Number

## Description:

Given a column title as appear in an Excel sheet, return its corresponding column number.

## Solution:

```c++
class Solution {
public:
    int titleToNumber(string s) {
        int res = 0;
        for (const auto &c: s)
            res = c - 'A' + 1 + res * 26;
        return res;
    }
};
```
